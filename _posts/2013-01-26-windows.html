还记得你的 Windows 是什么时候安装的么？

秀一下吧，开始 > 运行（快捷键 Win+R）:
英文系统：
cmd /k systeminfo | find “Original Install Date”
中文系统：
cmd /k systeminfo | find “初始安装日期”
update:请将双引号更换为英文输入状态下的双引号，WP 自动替换成了全角
注：直接运行 systeminfo 还有许多其他信息

国外一个网站，可以让你直接在浏览器上运行输入的 Java 8 代码

不过有一些限制：
  • 不能在代码中使用包名
  • 增加所需的所有 import 语句
  • 类名可以是任意有效的 Java 类名，但必须使用 public 修饰符
  • 每个类都应该有 main 方法，网站会调用这个方法
  • 程序必须在一分钟内结束运行，否则系统将终止执行
访问该网站：http://tryjava8-t20.rhcloud.com/

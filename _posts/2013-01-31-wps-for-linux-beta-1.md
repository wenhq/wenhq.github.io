[WPS For Linux Beta
1](http://linuxtoy.org/archives/wps-for-linux-beta-1.html):\
金山公司在今天发布了 WPS For Linux Beta 1。\
\
WPS 的  Linux 版本有以下特点：\
\

-   实时备份
-   Ubuntu 全局菜单
-   自动转换 Windows 格式的路径
-   中键剪贴板（全局选择剪贴板）
-   针对主流发行版提供相应安装包（DEB，RPM）
-   提供绿色版，直接解压即可运行
-   针对 Linux 复杂的系统环境提供环境检查机制并给出解决指导
